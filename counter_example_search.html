Counter Example Search#

This file contains a simple search function to check solutions for Fermat’s Last Theorem#

Recall that the statement of the problem is that there is no integer solutions to the equation below for positive integers a, b and c, and for \(n>2\)

\[ a^n + b^n = c^n \]
#This function will produce a sentence is a counter example of Fermat's Last Theorem is found
def check_fermat(a, b, c, n):
    if n > 2 and (a**n + b**n == c**n):
        print("You should be given a Nobel Prize")

This loop below searches for a counter example of the Fermat’s Last Theorem over all values of a, b and c between 1 and 10, and for a value of n between 3 and 100

for a in range (1,10):
    for b in range (1,10):
        for c in range (1,10):
            for n in range (3,100):
                check_fermat(a, b, c, n)

As we can see, the function returns nothing, which means there does not exist a counter example to the Fermat’s Last Theorem over the ranges defined above. Feel free to change the values above for a chance of the greatest discovery in the post pandemic era.

Let’s try again, with some different numbers

for a in range (10,20):
    for b in range (10,20):
        for c in range (10,20):
            for n in range (100,200):
                check_fermat(a, b, c, n)

Again, no output from the function, indicating that we are not getting a Nobel Prize any time soon.
